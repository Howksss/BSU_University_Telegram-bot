Это - код проекта телеграм-бота для абитуриентов Байкальского Государственного Университета.



---USER GUIDE---
Этот файл был создан для упрощения работы с проектом BGU_FAQ_BOT.

Установка: 
1) Откройте консоль( введите "cmd" в поисковике windows)
2) Внутри консоли введите команду "cd  Ваш\Путь\К\Папке\BGU_FAQ_BOT"
3) Внутри консоли введите команду "pip install requirements.txt"

Запуск:
1) Внутри консоли введите команду "cd  Ваш\Путь\К\Папке\BGU_FAQ_BOT"
2) Внутри консоли введите команду "python main.py"
3) Важно оставлять консоль открытой и не вводить никаких других комманд

Изменение списка администраторов: 
Список действующих администраторов находится в файле ".env", чтобы добавить пользователя в администраторы, вам необходимо:
1) Получить telegram-id нужного пользователя(в интернете куча способов это сделать) 
2) Скопировать его telegram-id(последовательность из 10 цифр)
3) В файле .env добавить новую строчку "ADMIN2 = скопированный номер пользователя" (Цифра после слова ADMIN должна меняться, в зависимости от количества действующих администраторов) 
4) Сохранить файл и перезапустить проект 

Просмотр Базы Данных: 
Бот сохраняет всех пользователей, когда-либо отправлявших команду /start(стартовая команда) боту и не отписавшхися от него. В базе данных сохраняется telegram-id пользователя и его имя в телеграмме. Для просмотра файла базы данных следует установить SQLiteStudio или любой другой редактор SQL файлов, после чего открыть файл "mydatabase.db", выбрать таблицу "users" и перейти во вкладку "Данные".
